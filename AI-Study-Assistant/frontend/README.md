# Frontend (Reference)

This project uses Flask templates from the top-level `templates/` folder and static files from `static/`.

The files in this `frontend/` folder are **optional reference copies** of the UI pages, useful if you want to host a separate frontend later.

For the working app, run the backend and open:
- `http://127.0.0.1:5000/login`
- `http://127.0.0.1:5000/register`
- `http://127.0.0.1:5000/dashboard`
